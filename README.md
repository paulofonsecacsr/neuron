__Neuron__ é um jogo de enigmas em tabuleiro físico controlado por sensor de voz, permitindo-lhe ser jogado por pessoas com deficiências motoras graves. Sua criação foi motivada por um projeto de primeiro período da faculdade CESAR School, o qual gerou forte interesse em nossa equipe de desenvolver algo único e acessível. Seu funcionamento é baseado em programação em C++ associada a Arduino Mega, sensor de voz Vosk Model desenvolvido em Python, ambos conectados por meio da biblioteca PySerial, e o site com os enigmas do jogo desenvolvido em HTML, CSS e JavaScript.

## Características
* __4 jogadores__: Ideal para pequenos grupos.
* __Dinâmica desafiadora__: Siga sua intuição para desvendar difíceis enigmas.
* __Controlado por comando de voz__:  Comandos atribuidos ao avanço do tabuleiro
* __Acessibilidade e Inclusão__: Desenvolvido para incluir jogadores com deficiêncas motoras graves
